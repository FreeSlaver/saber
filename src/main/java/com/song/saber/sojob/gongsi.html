公司介绍

优mall是什么？

优mall（优点联盟），诞生于互联网+时代，以振兴实体商业为己任，以让人们重新爱上逛街为目标，基于移动互联网的本地生活多边平台（顾客 - 店铺 - 购物中心）。让顾客享受优惠和便捷的同时提升其消费体验；让店铺、购物中心省钱和赚钱的同时更能抓住用户的心。


愿景：让人们重新爱上逛街。

文化：诚信为本，积极主动（主动，激情，合作与担当，拥抱变化），客户至上

团队：团队成员全部来自优秀的互联网公司（腾讯、百度、去哪儿等），优秀的团队，开放的文化以及个性化的发展机会


我们有：

乐观向上小而美的团队；

精致的薪酬福利体系；

友善的团队氛围


希望你是：

乐观向上富有激情

责任当担主动推进

信任勇敢相互扶持


更期望你再投出简历之前思考清楚：

a 如果加入一家创业公司是一条披荆斩棘又富有荣耀的路，你愿意吗？

b 每个人在团队中都要独挡一面，那你自己的独有价值会是什么呢？

c 这份工作，你期望收获些什么？
